# Strategy UI Assignment

This project is a responsive UI built using **HTML, CSS, and Vanilla JavaScript** as part of a web developer assignment.

## Features
- Market view tabs (Bullish, Bearish, Rangebound, Volatile)
- Date-based filtering using dropdown
- Dynamic strategy rendering
- Strategy count handling
- Empty state handling
- Responsive design (no frameworks used)

## Tech Stack
- HTML
- CSS
- JavaScript (Vanilla)

## Notes
- No external libraries or frameworks were used.
- Placeholder strategy values (e.g., Strategy1, Strategy2) are filtered out as per expected output.

## How to Run
Simply open `index.html` in any modern browser.
